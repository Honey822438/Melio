Pakistan's premium local products marketplace connecting artisans and small vendors with buyers across Pakistan.

![Melio Hero](docs/screenshots/hero.png)

## ✨ Features

- 🔐 Role-based auth — Buyer, Seller, Admin
- 🛍️ Product listings with filters, search, categories
- 🛒 Cart & multi-step checkout
- 📦 Order management & tracking
- ⭐ Reviews & ratings
- 💳 Cash on Delivery + JazzCash + Stripe
- 🏪 Seller & Admin dashboards

## 🧱 Tech Stack

| Layer | Tech |
|---|---|
| Frontend | React + TypeScript (Vite) |
| Backend | Express.js + TypeScript |
| Database | Supabase (PostgreSQL) |
| Storage | Supabase Storage |
| Payments | COD + JazzCash + Stripe |
| Deploy | Vercel + Railway |

## 🚀 Getting Started

### Backend
```bash
cd backend
cp .env.example .env
# Add your Supabase credentials to .env
npm install
npm run dev
```

### Frontend
```bash
cd frontend
cp .env.example .env
# Add VITE_API_URL=http://localhost:5000
npm install
npm run dev
```
